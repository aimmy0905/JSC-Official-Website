## 🎁 模块一：热门产品

### 这是什么？
这个模块展示了8个精选的热门产品，让访客能快速了解你们提供的商品类型。

### 包含什么内容？
- **8个产品卡片**，每个卡片包括：
  - 产品图片区域（目前显示图标占位符）
  - 产品标题（如：智能蓝牙耳机、便携式充电宝等）
  - 产品价格（如：$29.99）
- **"查看更多产品"按钮**：点击可跳转到产品页面

### 样式特点
- 💻 **电脑上**：4个产品横向排列一行，共2行
- 📱 **平板上**：3个产品一行
- 📲 **手机上**：2个产品一行
- 📱 **小手机**：1个产品占满一行

### 如何更换产品？
打开 `index.html` 文件，找到"热门产品"部分（大约在308行附近），你会看到：

```html
<div class="product-card">
    <div class="product-image">
        <i class="fas fa-box"></i>
    </div>
    <div class="product-info">
        <h3 class="product-title">智能蓝牙耳机 - 无线降噪音乐耳机</h3>
        <p class="product-price">$29.99</p>
    </div>
</div>
```

**修改方法：**
1. 修改 `<h3>` 标签里的文字 → 更换产品名称
2. 修改 `<p>` 标签里的价格 → 更换价格
3. 如果有产品图片，可以替换 `<i class="fas fa-box"></i>` 为：
   ```html
   <img src="images/products/你的图片.jpg" alt="产品名称">
   ```

## 💬 模块二：客户反馈（横向滑动）

### 这是什么？
这个模块展示了6条来自真实客户的好评，增加网站的可信度和说服力。所有卡片横向排列，可以滑动查看。

### 包含什么内容？
每个反馈卡片包括：
- **引号装饰**：右上角有一个紫色的圆形图标（带引号符号）
- **客户评价**：客户说的话
- **客户信息**：
  - 头像（目前显示图标占位符）
  - 客户姓名（如：贝查、希蔻等）
  - 平台名称（如：亚马逊、SHOPIFY、eBay等）

### 样式特点
- 📜 **横向滚动**：所有卡片排成一行，可以左右滑动
- 🎯 **三种导航方式**：
  1. **点击箭头按钮**：左右两侧的紫色圆形按钮
  2. **点击小圆点**：底部的圆点指示器，点击直接跳转
  3. **触摸滑动**：手机/平板上用手指左右滑动
- ✨ **隐藏滚动条**：界面干净整洁，没有难看的进度条
- 🔵 **小圆点指示器**：
  - 显示总共有多少个卡片（6个圆点）
  - 当前查看的卡片会变成长条形
  - 点击任意圆点可直接跳转
  - 滑动时自动更新激活状态
- 📏 **卡片尺寸**：
  - 电脑：350px宽
  - 平板：320px宽
  - 手机：280px宽
  - 小手机：260px宽
- 🎨 **悬停效果**：鼠标移上去时，卡片会向上浮动，显示阴影
- 💜 **装饰元素**：右上角的引号图标有独特的"水滴"形状
- ⬅️➡️ **导航按钮**：左右两侧有紫色圆形按钮，点击切换卡片

### 如何更换客户反馈？
打开 `index.html` 文件，找到"客户反馈"部分（大约在386行附近），你会看到：

```html
<div class="testimonial-card-compact">
    <div class="quote-icon-compact">
        <i class="fas fa-quote-right"></i>
    </div>
    <p class="testimonial-text-compact">服务美好，支持也不错......</p>
    <div class="customer-info-compact">
        <div class="customer-avatar-compact">
            <i class="fas fa-user"></i>
        </div>
        <div class="customer-details-compact">
            <h4 class="customer-name-compact">贝查</h4>
            <p class="customer-platform-compact">亚马逊花园</p>
        </div>
    </div>
</div>
```

**修改方法：**
1. 修改 `<p class="testimonial-text-compact">` 里的内容 → 更换客户评价
2. 修改 `<h4 class="customer-name-compact">` → 更换客户姓名
3. 修改 `<p class="customer-platform-compact">` → 更换平台名称
4. 如果有客户头像，可以在 `<div class="customer-avatar-compact">` 里添加：
   ```html
   <img src="images/avatars/客户头像.jpg" alt="客户姓名">
   ```

### 如何控制滑动？
客户反馈区域支持三种导航方式：

1. **点击箭头按钮** ⬅️➡️
   - 左右两侧有紫色圆形按钮
   - 点击向左/向右切换卡片

2. **点击小圆点** 🔵
   - 底部有6个小圆点
   - 当前卡片的圆点会变成长条形（紫色）
   - 点击任意圆点直接跳转到对应的卡片
   - 滑动时圆点会自动更新

3. **触摸滑动** 👆
   - 在手机/平板上用手指左右滑动
   - 支持流畅的触摸手势

**优化说明：**
- ✅ 已隐藏滚动条，界面更美观
- ✅ 移除了鼠标拖拽功能，使用更优雅的圆点导航
- ✅ 所有功能都已自动配置，无需设置！

## 📱 响应式设计

这两个模块在不同设备上的显示效果：

| 设备类型 | 热门产品 | 客户反馈卡片宽度 |
|---------|---------|-----------------|
| 💻 电脑（>968px） | 4列布局 | 350px宽 |
| 📱 平板（768-968px） | 3列布局 | 320px宽 |
| 📲 手机（480-768px） | 2列布局 | 280px宽 |
| 📱 小手机（<480px） | 1列布局 | 260px宽 |

**注意**：客户反馈在所有设备上都是横向滑动的，只是卡片大小会根据屏幕尺寸调整。

## 🔧 技术细节

### 使用的CSS类
- `.hot-products-section` - 热门产品区域
- `.hot-products-grid` - 产品网格布局
- `.testimonials-section` - 客户反馈区域
- `.testimonials-slider-container` - 滑动容器（包含按钮）
- `.testimonials-slider` - 可滑动的卡片区域
- `.testimonial-card-compact` - 单个反馈卡片（紧凑版）
- `.quote-icon-compact` - 引号装饰图标
- `.customer-info-compact` - 客户信息区域
- `.slider-btn` - 左右滑动按钮

### 使用的JavaScript函数
- `scrollTestimonials(direction)` - 控制滑动方向
  - 参数 `'prev'` - 向左滑动
  - 参数 `'next'` - 向右滑动
- `goToTestimonial(index)` - 跳转到指定卡片
  - 参数 `index` - 卡片索引（0-5）
- `updateDots()` - 更新圆点激活状态（自动调用）

### 图标库
使用了 Font Awesome 6.4.0 图标库：
- `fa-box` - 产品占位图标
- `fa-quote-right` - 引号图标
- `fa-user` - 客户头像占位图标

## 📝 常见问题

### Q1: 为什么产品显示的是图标而不是图片？
**答：** 这是因为目前使用的是占位符。等你准备好真实的产品图片后，就可以替换掉这些图标了。

### Q2: 可以显示少于8个产品吗？
**答：** 当然可以！如果你只想显示6个产品，只需要在HTML里删除不需要的产品卡片即可。

### Q3: 客户反馈可以增加到10个吗？
**答：** 可以！只需要复制一个反馈卡片的代码，修改内容后粘贴到反馈区域里就行了。

### Q4: 引号图标的颜色可以改吗？
**答：** 可以！在 `css/style.css` 文件里找到 `.quote-icon-compact`，修改 `background` 属性就能改变颜色。

### Q5: 如何调整产品价格的显示样式？
**答：** 在 `css/style.css` 文件里找到 `.product-price`，可以修改字体大小、颜色等属性。

### Q6: 滑动按钮可以隐藏吗？
**答：** 可以。即使隐藏按钮，还是可以用小圆点导航或触摸滑动。在CSS里找到 `.slider-btn` 添加 `display: none;` 即可。

### Q7: 可以同时显示更多卡片吗？
**答：** 可以！在CSS里修改 `.testimonial-card-compact` 的 `min-width` 和 `max-width`，把宽度改小，屏幕上就能同时显示更多卡片。

### Q8: 热门产品为什么和产品页面一样了？
**答：** 这是为了保持内容一致性。这样访客在首页看到的产品，点击进入产品页面后也能找到，避免混淆。如果你想在首页显示不同的产品，可以手动修改产品信息。

### Q9: 小圆点是做什么的？
**答：** 小圆点有两个作用：
1. **显示位置**：让你知道现在看的是第几个卡片
2. **快速导航**：点击圆点可以直接跳到对应的卡片

当前查看的卡片对应的圆点会变成长条形（紫色），其他的是小圆形（灰色）。

### Q10: 为什么看不到滚动条了？
**答：** 这是优化后的效果！滚动条已经被隐藏，使用更美观的小圆点指示器来显示位置。这样界面看起来更干净、更专业。

### Q11: 可以让卡片自动轮播吗？
**答：** 可以！在 `js/main.js` 里添加自动轮播代码。不过目前的设计是让用户主动控制浏览，体验更好。
