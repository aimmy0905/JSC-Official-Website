## 📖 这是什么？

这是一个**企业官网项目**，就像给公司做一张"网络名片"。

想象一下：
- 当别人想了解 JSC 公司时，就可以访问这个网站

## 🎯 网站包含哪些页面？

我们要做 **4个页面**：

### 1️⃣ **首页** 
- 作用：让访客第一眼就知道"我们是干什么的"
- 内容：公司介绍、核心服务、合作流程、优势展示

### 2️⃣ **服务页**
- 作用：详细介绍我们提供的服务
- 内容：品牌定制、产品采购、质量检查、物流合作

### 3️⃣ **产品页**
- 作用：展示热门商品
- 内容：商品图片、价格、搜索功能

### 4️⃣ **常见问题页**
- 作用：回答客户经常问的问题
- 内容：FAQ问答（待补充）

**1. 卡片尺寸精准调整：** 📐
- 卡片宽度调整为 **340px**（最佳展示宽度）
- 卡片间距调整为 **1.5rem**（24px）

- 计算公式：容器宽度1200px - 左右按钮空间(60px×2) = 1080px 可用宽度
