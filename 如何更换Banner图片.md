# 📸 如何更换首页Banner图片

## 🎯 简单3步，轻松更换Banner图片！

---

## 步骤1：准备你的图片 🖼️

### 图片要求：
- **格式：** PNG、JPG 或 JPEG
- **建议尺寸：** 宽度 800-1200px，高度 600-900px
- **建议比例：** 4:3 或 16:9
- **文件大小：** 建议不超过 2MB（太大会影响加载速度）

### 图片内容建议：
- 仪表板截图（数据统计图表）
- 产品展示图
- 办公场景图
- 或任何能代表你业务的图片

---

## 步骤2：放置图片到正确位置 📂

### 方法A：使用文件夹（推荐）

1. **打开项目文件夹**
   ```
   /Users/aimmy/Desktop/jsc官网设计/
   ```

2. **进入图片文件夹**
   ```
   images/banner/
   ```

3. **把你的图片复制进去**
   - 图片名称改为：`dashboard.png` 或 `dashboard.jpg`
   - 或者使用你自己的名称（需要修改代码，见下方）

---

### 方法B：使用自定义文件名

如果你想用自己的文件名（比如 `my-banner.jpg`）：

1. 把图片放到 `images/banner/` 文件夹
2. 打开 `index.html` 文件
3. 找到第50行：
   ```html
   <img src="images/banner/dashboard.png" alt="JSC Dropshipping Dashboard" class="banner-img">
   ```
4. 修改为：
   ```html
   <img src="images/banner/my-banner.jpg" alt="JSC Dropshipping Dashboard" class="banner-img">
   ```

---

## 步骤3：查看效果 👀

### 查看网站：

1. **保存所有修改**（如果改了文件名）

2. **刷新浏览器**
   - Mac: 按 `Cmd + R`
   - Windows: 按 `Ctrl + R`
   - 或者重新双击打开 `index.html`

3. **查看Banner区域**
   - 你的图片应该显示在右侧
   - 鼠标放上去会有轻微放大效果

## 🔧 常见问题解决

### Q1: 图片不显示怎么办？

**检查清单：**
- [ ] 图片文件确实在 `images/banner/` 文件夹中
- [ ] 文件名和代码中的名称完全一致（注意大小写）
- [ ] 图片格式是 PNG、JPG 或 JPEG
- [ ] 刷新了浏览器（按 Cmd+R 或 Ctrl+R）

**解决方法：**
1. 打开浏览器的开发者工具（按 F12）
2. 查看 Console 标签，看是否有错误提示
3. 检查图片路径是否正确

---

### Q2: 图片显示得太大或太小？

**调整方法：**

打开 `css/style.css`，找到第250行左右：

```css
.dashboard-mockup .banner-img {
    max-width: 100%;     /* 可以改成 80% 或其他值 */
    height: auto;
}
```

调整 `max-width` 的值：
- `100%` = 填满整个区域
- `80%` = 占80%的宽度
- `500px` = 固定500像素宽度

---

### Q3: 能用网络图片吗？

**可以！** 如果图片在网上，直接用网址：

修改 index.html 第50行：
```html
<img src="https://你的图片网址.com/image.jpg" alt="JSC Dropshipping Dashboard" class="banner-img">
```

**注意：** 网络图片需要确保：
- 图片网址可以正常访问
- 网站允许跨域使用图片
- 图片网站稳定（不要随时失效）

---

### Q4: 如何优化图片？

**建议使用图片压缩工具：**
- **在线工具：** tinypng.com（免费，效果好）
- **Mac软件：** ImageOptim（免费）
- **通用软件：** Photoshop（导出时选择"网页用"）

**优化好处：**
- 加载速度更快
- 节省流量
- 提升用户体验
